Position

Relative

의미 : 자기 자신을 기준으로 이동

position: relative;
  • 원래 위치 기준으로 top, bottom, left, right 값에 따라 결정
  • 위치를 이동하면서 다른 요소에 영향을 주지 않음.
  • 문서 상 원래 위치 그대로 유지

Absolute

부모 요소를 기준으로 배치 (단, 부모에 relative가 없을 경우, body가 기준)

position: absolute;

fixed

viewport 기준

position: fixed;

sticky

scroll 영역 기준 배치

position: sticky;
